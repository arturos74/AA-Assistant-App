# ArcheAge Assistant App

## Overview
**ArcheAge Assistant App** is an Android application currently in development that aims to provide players of the MMORPG *ArcheAge* with helpful in-game tools and calculators. The goal of the project is to create a centralized mobile utility that simplifies common gameplay decisions such as currency conversion and skill planning.

This project is primarily a **learning and portfolio project**, focused on improving Android development skills, UI design, and Kotlin programming.
